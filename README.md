
# Project Setup Guide

## Server Setup

1. Install `nodemon`:
   ```bash
   npm install nodemon
   ```

2. Install server dependencies:
   ```bash
   npm install
   ```

3. Start the server:
   ```bash
   npm run dev
   ```

## Client Setup

1. Navigate to the `client` directory:
   ```bash
   cd client
   ```

2. Install `vite`:
   ```bash
   npm install vite
   ```

3. Install client dependencies:
   ```bash
   npm install
   ```

4. Start the client:
   ```bash
   npm run dev
   ```
